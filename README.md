"# book_completer" 

To Run:

# Install Dependencies
npm install --save

# Build with Babel
npm run-script build 

# Start

node lib\index.js

Args:

-i, --inputfile [file] Input CSV File
-l, --limit <n> How many records should be read
-o, --offset <n> How many records should be skipped
-f, --outputfile [file] Output CSV File
